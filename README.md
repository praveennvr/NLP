# NLP
The file contains movie review classification using following:
  1. Webscraping
  2. Text processing, cleaning
  3. Spell correction
  4. Translation
  5. TF-IDF Vectorization
  6. Classification using SVD
  7. Prediction of rating using
      a. Linear Regression
      b. Linear SVC
      c. Random Forest
      d. XGBoost
      e. CNN + RNN (LSTM)
